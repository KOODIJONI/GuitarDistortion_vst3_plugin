# How it works
This plugin works by using a soft-clip function on the userâ€™s audio signal. Soft-clipping changes the sound by gently cutting the highest peaks of the signal, which creates a type of distortion.

Unlike hard-clipping, which cuts the sound sharply, soft-clipping makes the peaks rounder and smoother. This gives the sound a warmer and more natural feel, instead of making it harsh.

You can control how strong the effect is using gain knob â€” from a light, warm touch to a heavier, more distorted sound.
